This is a distribution of openmpi binaries, compiled for Mac OS X by Peter Willendrup, based on hpc.sourceforge.net gcc and gfortran.

openmpi-1.6.5.tar.bz2 is the original sourcecode which has been compiled from the source folder using

./configure --disable-mpi-f77 --disable-mpi-f95 --prefix=/opt/mccode/openmpi

Installs into the common mccode install location /opt/mccode/openmpi
