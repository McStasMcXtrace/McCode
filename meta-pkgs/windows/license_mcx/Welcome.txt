Welcome to the McXtrace installer on Windows

This installer allows you to install essential McXtrace and Python-based tools, plus a miniconda3 python including a compiler.

Other recommended software is
	•	NeXus and MPI binaries from the ‘extras’ folder of the McXtrace download page
	•	iFit from http://ifit.neutroncode.org
	•	InstantPlayer X3D/VRML viewer from http://www.instantreality.org/downloads/
