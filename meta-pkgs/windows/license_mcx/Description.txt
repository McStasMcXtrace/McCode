Info on install configuration on Windows:

Dependencies:
	•	All essential dependencies (also a miniconda3 Python) are included in the meta-package.
	•	You may optionally also install the Strawberry Perl bundle provided in your downloaded McXtrace archive

A default installation will bring you:
	•	mcxtrace (core code-generator)
	•	mcxtrace-comps (components
	•	mcxtrace-manuals
	•	mcxtrace-tools-python-mccodelib
	•	mcxtrace-tools-python-mxrun (Python-based mxrun tool)
	•	mcxtrace-tools-python-mxdoc (Python-based mxdoc tool)
	•	mcxtrace-tools-python-mxplot-pyqtgraph (Python-based mxplot tool)
	•	mcxtrace-tools-python-mxplot-matplotlib (Python-based mxplot tool)
	•	mcxtrace-tools-python-mxdisplay-webgl (Python-based mxdisplay tool)
	•	mcxtrace-tools-python-mxdisplay-pyqtgraph (Python-based mxdisplay tool)
	•	mcxtrace-tools-python-mxdisplay-matplotlib (Python-based mxdisplay tool)
	•	mcxtrace-tools-python-mxgui (Python-Qt/Scintilla based mxgui tool)
	•	mcxtrace-tools-perl-commandline (Legacy set of Perl-based cmdline tools)
	•	mcxtrace-tools-perl (Legacy set of Perl-based gui tools)
